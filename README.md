# ST558 Project 2

This repository contains the second project for **ST558**, creating an interactive R Shiny Application that allow users to explore a Banking - Marketing Targets Datset available on [Kaggle](https://www.kaggle.com/datasets/prakharrathi25/banking-dataset-marketing-targets). The goal is to gain a better understanding of R Shiny development, reactive programming, and data visualization while producing a user-friendly interface that allows for the data to be segmented and explored by everyone.

## Project Overview

- Download and process data for exploration in Shiny.
- Perform EDA to investigate different variables available in the dataset.
- Create and R Shiny app that allows user to segment the data into subsets for investigation.
- Provide user with graphical representations and other helpful statistics to better understand data.

## Goals

- Demonstrate proficiency in R Shiny.
- Create a user-friendly app for exploring the data.
- Produce clear and reproducible code and documentation.
